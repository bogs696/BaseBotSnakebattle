Это мой скелет бота для игры "snakebattle". За основы был взят один случайно найденный на просторах сервера AI (https://github.com/codenjoyme/codenjoy/tree/master/CodingDojo/games/snake-battle) и переделанный под свои нужды.
Одним из изменений класс Elements был перенесен в папку с классом YouSolver так как класс не хотел отображаться.
В данном боте реализованы следующие функции:
- Определять тупиковые точки на карте;
- Определять время действия пилюль;
- Определять какой объект был съеден;
- Вести подсчет сколько камней хранится в змейке;
- Находить вражеских змеек на карте;
- Была реализована динамическая смена стратегий на игру (сейчас реализована только 1 стратегия на поедание яблок).
Для запуска змейки требуется вставить свой код и свою почту в url ссылку в методе main.
Для разработки своей логики предлагается к изменению следующее:
1) Добавить больше вариантов стратегий в enum Stratege (там только стратегия под названием "APPLE" которая просто пожирает яблоки).
2) В методе isBarrierAt() указать для стратегий какие объекты они должны избегать основываясь на текущей стратегии и своем состоянии.
3) В методе searchTarget() указывать в зависимости от состояния и стратегии какие объекты искать.
4) В методе chooseStratege() анализировать текущее положение борда и противников возвращать актуальную стратегию.

П.С. Реализация выполнена по большей части на коленке и выполнялась по принципу "главное запуститься и успеть отработать за <1 сек.", из-за этого во многих местах страдает читаемость кода.
